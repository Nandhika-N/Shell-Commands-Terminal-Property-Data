# Shell-Commands-Terminal-Property-Data

This project focuses on using the Mac shell command Terminal to perform data manipulation, data wrangling and analysis. Tasks such as reading and extracting data from files, applying various exploration and visualisation techniques for analysis, summarising as well as discussing and communicating findings.

The project involves:
- Reading and extracting data from files.
- Applying various exploration and visualisation techniques for analysing data summarising.
- Discussing and communicating findings using Shell.
